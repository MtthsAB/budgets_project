📋 Como Testar as Funcionalidades

Teste 1: Múltiplos Tamanhos por Módulo

  1. Acesse: Novo Orçamento
  2. Clique em "Adicionar Item" → Selecione "Sofá"
  3. Escolha um sofá (ex: Le Coultre)
  4. Marque um módulo (ex: MOD 08 PUFE TERMINAL)
  5. Marque múltiplos tamanhos do mesmo módulo
  6. Defina quantidades diferentes para cada tamanho
  7. Verifique se os subtotais aparecem corretamente

Teste 2: Botão "Adicionar Item"

  1. Configure um sofá com módulos e tamanhos
  2. Adicione alguns acessórios opcionais
  3. Clique em "Adicionar Item"
  4. Verifique se o item aparece na lista do pedido
  5. Confirme se o preço total está correto

Teste 3: Funcionalidades de Remoção

  1. Configure módulos com múltiplos tamanhos
  2. Use botão ❌ para remover tamanhos específicos
  3. Use botão 🗑️ para remover módulos completos
  4. Verifique atualizações automáticas do resumo

⚠️ Pontos de Atenção

⚠️ Certifique-se de que ao menos um módulo tem tamanho selecionado antes de adicionar ao pedido
⚠️ Verifique se os cálculos de preço estão corretos para múltiplas quantidades
⚠️ Teste a remoção de tamanhos para garantir que não quebra a estrutura
⚠️ Confirme que os dados são enviados corretamente para o backend
